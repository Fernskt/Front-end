Calculadora de sueldos

Ésta aplicación web ayuda a los trabajadores de OCA LOG S.A. a calcular su sueldo neto y bruto con precisión y eficiencia. La aplicación utiliza tecnologías web modernas como HTML5, CSS, Bootstrap y JavaScript y está actualizada con las últimas regulaciones fiscales y laborales para garantizar la precisión de los cálculos en todo momento. Además, la aplicación cuenta con una interfaz intuitiva y fácil de usar, y se han aplicado técnicas de experiencia de usuario para optimizar su efectividad.
